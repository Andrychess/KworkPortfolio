# Портфолио

Статический сайт (`index.html` + `Works/`). Хостится на Git без сервера.

## Структура работ

```
Works/
  Категория/           → фильтр
    Название плитки/   → карточка
      *.png / *.jpg
      *.pdf
      *.pptx
```

## Как обновлять

1. Положите файлы в `Works/...`
2. Запустите сборку:

```bash
python _sync.py
```

или `Собрать сайт.bat`

3. Закоммитьте изменения (`index.html`, `Works/`, превью `.previews/`) и запушьте.

Сайт открывается как обычный статический HTML (GitHub Pages и т.п.).
